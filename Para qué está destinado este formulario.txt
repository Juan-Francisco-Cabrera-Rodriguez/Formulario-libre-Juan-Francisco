Esta destinado para las opiniones y aportaciones de los usuarios de mi paguina web

Etiqetas HTML
<!DOCTYPE>: Define el tipo de documento 
<html>: Define la raíz de un documento HTML
<head>: Contiene metadatos / información del documento. 
<meta>: Define metadatos sobre un documento HTML
<link>: Define la relación entre un documento y un recurso externo (más utilizado para vincular a hojas de estilo) 
<title>: Define un título para el documento 
<body>: Define el cuerpo del documento
<div>: Define una sección en un documento 
<form>: Define un formulario HTML para la entrada del usuario
<h1>: Define encabezados HTML
<label>: Define una etiqueta para un elemento <input>
<input>: Define un control de entrada 
<textarea>: Define un control de entrada de varias líneas (área de texto)

HTML Atributos
class: Especifica uno o más nombres de clase para un elemento (se refiere a una clase en una hoja de estilo)
id: Especifica una identificación única para un elemento 
lang: Indica el idioma del elemento 
charset: juegos de caracteres
action: La URL del programa que procesa la información enviada en el formulario.
name: Nombre del elemento. Por ejemplo utilizado por el servidor para identificar los campos en el envio de formularios.
content: Un valor asociado con http-equiv o name dependiendo del contexto.
rel: Especifica la relación entre el objeto destino y el objeto enlace.
href: La URL de un recurso asociado.
for: Describe elementos que pertenecen a éste.
type: Define el tipo de elemento.
placeholder: Proporciona una sugerencia al usuario de lo que puede ser introducido en el campo.
value: Define el valor predeterminado a ser mostrado al cargar la página.
